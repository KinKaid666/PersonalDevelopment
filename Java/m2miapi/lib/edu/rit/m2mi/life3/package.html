Package edu.rit.m2mi.life3 is a program for John Horton Conway's Game of Life using M2MI. Class Life is the main program. To run the program, execute this command, specifying the desired number of rows and columns (16 by 16 in this example):

java edu.rit.m2mi.life3.Life 16 16

The program displays a GUI:

Click on a cell to toggle the cell between live and dead. Click on the Start, Stop, Step, and Reset buttons to control the simulation.

By default, cells in the top row are adjacent to cells in the bottom row, and cells in the left column are adjacent to cells in the right column (toroidal topology). Specify the -p flag to make opposite edges not adjacent (planar topology):

java edu.rit.m2mi.life3.Life -p 16 16
